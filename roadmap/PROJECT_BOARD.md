# GitHub Project / Board

Proponowana nazwa projektu:

```text
Hudquarna Drukarka - Progress
```

## Kolumny / statusy

1. **Backlog** - pomysły i zadania jeszcze nie ruszone.
2. **Ready** - zadania gotowe do pracy.
3. **In progress** - aktualnie robione.
4. **Blocked** - czeka na decyzję, część, test lub informację.
5. **Review / Test** - wymaga sprawdzenia.
6. **Done** - zakończone.

## Pola dodatkowe

- **Area:** elektronika, mechanika, firmware, integracja, dokumentacja, testy.
- **Priority:** P0, P1, P2, P3.
- **Target:** Faza 0, Faza 1, Faza 2, Faza 3, Faza 4.
- **Date / update:** data ostatniej zmiany.

## Zasada pracy

Każde większe zadanie powinno mieć issue, a każdy dzień powinien mieć wpis w `progress/` z linkami do issue i dodanych plików.
